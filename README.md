# 🎬 Movie Data Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing a movie dataset to uncover insights about movie trends, genres, and audience engagement using Power BI.

---

## 🔍 Key Insights

* Movie releases have increased over time 📈
* Drama and Comedy are the most dominant genres 🎭
* Audience engagement varies across genres
* Some genres contribute more to overall popularity

---

## 🛠 Tools & Technologies

* Python (Exploratory Data Analysis)
* Power BI (Data Visualization)

---

## 📊 Dashboard Features

* Movie release trends over time
* Genre-wise comparison
* Vote count and popularity analysis
* Interactive visualizations

---

## 📂 Files Included

* `cleaned_movie_data.csv`
* `dashboard.pbix`
* Dashboard screenshot

---

## 🚀 How to Use

1. Download the dataset
2. Open the `.pbix` file in Power BI
3. Explore the interactive dashboard
